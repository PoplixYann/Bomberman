Description: 
Ce jeu est basé sur Bomberman une license qui compte bon nombre de jeux , celui-ci n'est pas totalement une copie car j'y ai ajouté 
quelques touches personnel notamment car je n'ai pas jouer au jeu original, dans mon Bomberman vous pourrez soit jouer en solo où le but
est de finir les 10 niveaux disponibles, pour passer un niveau ils vous faudra tuez tous les ennemies présents sur la map pour ensuite
prendre la porte amenant au niveau suivant. Soit en mode multijoueur de 2 à 4 joueurs bien qu'il ne soit pas terminer, le but 
du multi est d'être le dernier survivant parmis tous les joueurs.

Contrôles:
	- Menu :
		-Fleches : Déplacement
		-Entrée : Confirmer
		-Echap : Retour arrière
		-F1 : Permet de lancer le mode solo soit au niveau 1, niveau 10 (dernier niveau)
			  , niveau 11 (niveau test permettant de tester l'écran de victoire). Un indicateur est present dans le menu principal en solo
			  
	- En jeu :
		-En solo :
			(Touches peuvent être modifier dans les options keybindings)
			-z,q,s,d : Déplacement
			-Espace : Poser bombes
			-E : Utiliser détonateur (si vous l'avez)
			-Echap : Ouvrir menu pause
		-En multi :
			Joueur 1 :
				-z,q,s,d : Déplacement
				-Espace : Poser bombes
			Joueur 2 :
				-Fleches : Déplacement
				-R-Shift : Poser bombes
			Joueur 3 :
				-i,j,k,l : Déplacement
				-m : Poser bombes
			Joueur 4 :
				-8,4,5,6(Pav num) : Déplacement
				-0(Pav num) : Poser bombes
			Echap : Retour a lobby


Fonctionnalités:
   - Animations dans les menus
   - Simulation d'une camera si la map est trop grande
   - Options Audio et Keybinding
   - Sauvegarde des options et du skins choisi
   
Bugs connus:
